# Pokemon list
This app used:
- React/Redux
- Recompose
- Semantic UI
- api for get nice pokemons

## How to install
Simply clone this repo and install all dependencies using ``` npm install ```.
 
## How to start
Start app by typing ``` npm start ```.
Pokemon list will run at ``` :8080 ``` port
